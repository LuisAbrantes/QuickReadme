# QuickReadme

A command-line tool to quickly generate README.md files.

# Installation

Clone the repository and install locally:

git clone https://github.com/LuisAbrantes/quickreadme.git <br> 
cd quickreadme <br> 
pip install -e . <br> 

# Usage

After installing, navigate to your project directory and run:

quickreadme

Follow the interactive prompts to input your projectâ€™s details. Once completed, a README.md file will be generated in your current directory with the information provided.
